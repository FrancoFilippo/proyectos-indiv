Mis Conocimientos.

HTML5

HTML5 ha sido una parte fundamental de mi conjunto de habilidades. A lo largo de mi experiencia, he cultivado un conocimiento sólido en esta tecnología, aprovechándola para crear sitios web dinámicos y compatibles con los estándares actuales.

CSS

Puedo organizar y diseñar páginas usando cajas, Flexbox y Grid. Agrego movimientos y transiciones para hacer las páginas más interactivas. Además, me aseguro de que los sitios se vean bien en diferentes dispositivos.

JavaScript

Con habilidades básicas en JavaScript, puedo crear interactividad en sitios web utilizando variables para almacenar datos, operadores para realizar cálculos simples, estructuras de control para tomar decisiones y bucles para repetir tareas.
